# pnRtp
Простой, но мощный плагин для рандомной телепортации игроков на серверах Minecraft.

## Особенности
- **3 режима телепортации:** `/rtp near`, `/rtp far`, `/rtp long` с разными радиусами и независимой настройкой кулдауна.
- **Поддержка Title и Subtitle:** Отправка заголовков на экран при успешной телепортации с настраиваемыми таймингами.
- **Безопасность:** Проверка безопасной локации перед телепортацией (избегание воды, лавы, деревьев, и поддержка WorldBorder).
- **Оптимизация:** Асинхронный поиск локации, что не нагружает основной поток сервера (TPS).

## Команды и права
- `/rtp <near|far|long>` - Телепортироваться в случайное место (Право: `pnrtp.use`)
- `/rtp reload` - Перезагрузить конфигурацию плагина (Право: `pnrtp.admin`)

## Конфигурация (config.yml)
Все параметры гибко настраиваются в файле `config.yml`. Вы можете изменять радиус и кулдаун для каждого из режимов по отдельности, а также кастомизировать все сообщения.
